数学 / The math

注意力对一组 query 在一组 key-value 上做加权聚合:

Attn(Q,K,V)=softmax ⁣(QK⊤dk)V\mathrm{Attn}(Q,K,V) = \mathrm{softmax}\!\left(\frac{QK^\top}{\sqrt{d_k}}\right)VAttn(Q,K,V)=softmax(dk​​QK⊤​)V

  • QK⊤QK^\topQK⊤:每个 query 与每个 key 的点积相似度,形状 (Lq,Lk)(L_q, L_k)(Lq​,Lk​)。
  • dk\sqrt{d_k}dk​​ 缩放:设各维独立、均值 0 方差 1,则 Var(q⋅k)=dk\mathrm{Var}(q\cdot k)=d_kVar(q⋅k)=dk​。不缩放会把 softmax 推向饱和、梯度消失;除以 dk\sqrt{d_k}dk​​ 把方差拉回 ~1。
  • softmax 沿 key 维归一化 → 每个 query 的注意力分布。
  • 乘 VVV:用该分布对 value 加权求和。
  • 多头:把 dmodeld_{model}dmodel​ 切成 hhh 个 dhead=dmodel/hd_{head}=d_{model}/hdhead​=dmodel​/h 子空间,各自独立注意力 → 拼接 → 输出投影 WOW_OWO​,让不同头关注不同关系。
