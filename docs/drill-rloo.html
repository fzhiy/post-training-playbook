Ai=ri−1K−1∑j≠irj=ri−(∑krk)−riK−1A_i = r_i - \frac{1}{K-1}\sum_{j\neq i} r_j = r_i - \frac{(\sum_k r_k) - r_i}{K-1}Ai​=ri​−K−11​j=i∑​rj​=ri​−K−1(∑k​rk​)−ri​​

策略梯度损失(最小化):  L=−1BK∑isg(Ai) log⁡πθ(oi)\;L = -\frac{1}{BK}\sum_{i}\mathrm{sg}(A_i)\,\log\pi_\theta(o_i)L=−BK1​∑i​sg(Ai​)logπθ​(oi​),其中 sg\mathrm{sg}sg 是 stop-gradient(基线不回传)。

性质:每个 prompt 内 ∑iAi=0\sum_i A_i = 0∑i​Ai​=0(无偏、低方差基线)。
