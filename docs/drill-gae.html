数学 / The math

TD 残差:  δt=rt+γV(st+1)−V(st)\;\delta_t = r_t + \gamma V(s_{t+1}) - V(s_t)δt​=rt​+γV(st+1​)−V(st​)

GAE 是 δ\deltaδ 的指数加权和:

AtGAE(γ,λ)=∑l=0∞(γλ)l δt+lA_t^{\text{GAE}(\gamma,\lambda)} = \sum_{l=0}^{\infty}(\gamma\lambda)^l\,\delta_{t+l}AtGAE(γ,λ)​=l=0∑∞​(γλ)lδt+l​

实现上用反向递推(从后往前):  At=δt+γλ At+1\;A_t = \delta_t + \gamma\lambda\,A_{t+1}At​=δt​+γλAt+1​。回报 Rt=At+V(st)R_t = A_t + V(s_t)Rt​=At​+V(st​)。

两个极端:

  • λ=0\lambda=0λ=0:At=δtA_t=\delta_tAt​=δt​ —— 单步 TD,低方差、高偏差。
  • λ=1\lambda=1λ=1:At=∑lγlrt+l−V(st)A_t=\sum_l\gamma^l r_{t+l} - V(s_t)At​=∑l​γlrt+l​−V(st​) —— 蒙特卡洛,高方差、低偏差。
  • λ∈(0,1)\lambda\in(0,1)λ∈(0,1):在偏差/方差间插值。
