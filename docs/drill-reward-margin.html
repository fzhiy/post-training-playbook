数学 / The math

Bradley-Terry 模型:给定 chosen / rejected 的标量奖励 rw,rlr_w, r_lrw​,rl​,

P(w≻l)=σ(rw−rl)P(w \succ l) = \sigma(r_w - r_l)P(w≻l)=σ(rw​−rl​)

训练 reward model = 最大化该似然 = 最小化成对 logistic 损失:

L=− E[log⁡σ(rw−rl)]L = -\,\mathbb{E}\big[\log\sigma(r_w - r_l)\big]L=−E[logσ(rw​−rl​)]

直觉:把 chosen 的分数推高于 rejected,差距(margin)rw−rlr_w-r_lrw​−rl​ 越大,损失越小。

  • margin =0=0=0 → L=log⁡2≈0.693L=\log 2\approx 0.693L=log2≈0.693;
  • margin →+∞\to+\infty→+∞ → L→0L\to 0L→0。

诊断量:margin(平均奖励差)、accuracy(排序正确的比例 rw>rlr_w>r_lrw​>rl​)。

与 DPO 的关系

DPO 把策略本身当成隐式 reward(r=βlog⁡πθπrefr=\beta\log\frac{\pi_\theta}{\pi_{\text{ref}}}r=βlogπref​πθ​​),它的损失就是把这里的 rw,rlr_w,r_lrw​,rl​ 换成隐式奖励的同一个 BT 损失——所以 DPO「不需要单独的 reward model」。
